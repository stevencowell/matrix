How it works

Use this tool when you need to bulk update subjects. It translates CSV spreadsheets into JSON lineCells patches and can validate a patch that someone has shared with you.

  1. Download the Allocation Data Template CSV and fill in your subjects, teachers and lines.
  2. Upload the completed CSV (or another exported CSV) to preview the first 30 cells and generate the JSON patch automatically.
  3. Optional: paste an existing JSON patch into the panel on the right to validate it and download a tidy copy.
